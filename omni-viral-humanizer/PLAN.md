# omni-viral-humanizer — Standalone Product Plan

## What It Is

A single AgentSkill that takes any raw idea/draft/topic and outputs ready-to-post, humanized, algorithm-optimized content for X, LinkedIn, TikTok, Reddit, Instagram, and YouTube — individually or all at once.

**Not** part of the social-skills modular pipeline. This is the quick-fire tool that skips setup and just ships content.

## Positioning

| | social-skills (modular pack) | omni-viral-humanizer (standalone) |
|---|---|---|
| Setup | Foundation context interview required | Zero setup |
| Workflow | Multi-skill pipeline with gates and review | Single skill, one-shot |
| Depth | Full content operation (strategy → analytics) | Content generation only |
| Use case | "Build my content system" | "Make this post go viral right now" |
| Target | Ongoing content operations | Quick-fire content needs |
| Output | Platform-native content + calendar + analytics | Platform-native content + scoring |

## v1 Scope (Current)

- 3 platforms: X, LinkedIn, TikTok
- Humanization layer
- Per-platform optimization with algorithm-aware rules
- JSON + Markdown output contract
- Reference files per platform (algo deep dive + viral examples)
- Templates for each platform format
- Before/after examples
- Cross-posting adaptation notes

## v2 Scope (Next)

### Platform Expansion
- Add Reddit, Instagram, YouTube Shorts
- Per-platform reference files for each new platform
- Reddit requires special treatment (anti-promotional voice, subreddit-aware)

### Humanization Upgrade
- Expand AI-tell detection checklist with platform-specific cringe patterns
- Add "cringe score" to output (0-10 scale, what specifically sounds AI-generated)
- Platform-specific humanization (LinkedIn-AI-slop differs from Twitter-AI-slop)

### Viral Examples (Real Data)
- Replace placeholder examples with 5 real posts per platform
- Include: URL/screenshot, date, metrics snapshot, "why it worked" analysis
- Quarterly refresh cycle
- Source: manual curation from top creator accounts + SociaVault API pulls

### Engagement Scoring Upgrade
- Weighted scoring model per platform (currently heuristic, make it transparent)
- X scoring: hook strength (30%) + reply potential (25%) + scanability (20%) + algo fit (25%)
- LinkedIn scoring: dwell-time intro (30%) + save-worthiness (25%) + comment depth potential (25%) + format fit (20%)
- TikTok scoring: 3s hook (35%) + completion potential (25%) + share/duet potential (20%) + search alignment (20%)
- Reddit scoring: authenticity (35%) + value density (25%) + community fit (25%) + title strength (15%)
- Instagram scoring: share/send potential (30%) + save potential (25%) + visual hook (25%) + caption quality (20%)
- YouTube scoring: CTR potential (30%) + retention hook (25%) + search alignment (25%) + CTA strength (20%)

### Media Prompts Upgrade
- Platform-specific image generation prompts (dimensions, style, text overlay guidance)
- Video shot list suggestions for TikTok/Reels/Shorts
- Carousel slide-by-slide content outlines for LinkedIn/Instagram
- Thumbnail concepts for YouTube

### Output Format Enhancements
- Add `"platform_specific_warnings"` field (e.g., "This post mentions a link — move to self-reply on X")
- Add `"a_b_test_suggestion"` field (alternative hook or CTA to test)
- Add `"repurposing_notes"` field (how this content could be adapted further)

## v3 Scope (Future)

### Account-Aware Mode
- Optional: load a voice-profile.md (generated by account-analysis skill from the modular pack)
- Humanizer adapts to match the user's specific voice DNA instead of generic "human"
- Bridge between standalone and modular pack

### Trend Integration
- Optional: fetch trending topics/sounds before generating
- TikTok trending sounds awareness
- X trending topics / discourse graph awareness
- LinkedIn trending articles / newsletter topics

### Multi-Variant A/B Generation
- Generate 3-5 variants per platform (not just 1-2)
- Score and rank variants
- Suggest which to post first and which to hold for testing

### Scheduling Integration
- Output includes optimal posting times based on audience timezone
- Optional: generate ICS calendar entries for scheduled posts
- Optional: generate Buffer/Hootsuite/Later import format

## Distribution

## Quarterly Maintenance

- Refresh viral examples (5 per platform)
- Update algorithm deep dives with any platform changes
- Update humanization checklist with new AI-tell patterns
- Run test.py to validate structure
- Bump version in SKILL.md frontmatter

## Success Metrics

1. User can go from raw idea → ready-to-post content for 3+ platforms in under 2 minutes
2. Output passes "would a real human post this?" sniff test
3. Platform-specific adaptations are genuinely different (not copy-paste with minor tweaks)
4. Engagement scores correlate with actual performance (validate quarterly)
5. Zero fabricated claims in reference files
