## Usage Notes

- **X/Twitter**: Short, punchy. Curiosity + contrarian + data hooks perform best. First line IS the hook (no "see more" click).
- **LinkedIn**: Story + value + personal hooks dominate. First ~210 chars visible before "see more" — hook MUST land there.
- **Reddit**: Question + data + contrarian hooks work. Avoid anything that reads as self-promotional.
- **Instagram**: Pattern interrupt + story hooks for captions. Visual hook (first slide) matters more than text.
- **TikTok**: First 3 seconds = hook. POV + "this is your sign" + curiosity gaps work best. Spoken, not written.
- **YouTube**: Thumbnail + title = hook. Data + curiosity + contrarian hooks for titles. Keep under 60 chars.

## Anti-Pattern: Hook Stuffing

Don't use multiple hook formulas in the same post. One hook, committed to fully. Stacking hooks ("Unpopular opinion: I spent 6 months analyzing data and most people get this wrong") reads as AI-generated.
