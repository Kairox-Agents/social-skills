# Gotchas — Thread Writing

## 1. The "Thread When a Tweet Would Do" Problem

**What happens:** Agent writes a 7-tweet thread for an idea that could be expressed in a single punchy tweet. The thread has 3-4 filler tweets padding out a simple point.

**Why it matters:** Engagement drops ~60% after tweet 4 in a thread. If your point fits in one tweet, the thread format actively hurts you — most people won't make it to the valuable part.

**How to avoid:** Before threading, ask: "Does this idea have 5+ genuinely distinct points, or is it one idea with examples?" If the latter, write a single tweet. Not every topic deserves a thread.

## 2. Engagement Cliff After Tweet 4

**What happens:** Agent front-loads setup and context in tweets 1-3, then puts the real value in tweets 5-8. By tweet 5, 60%+ of readers are gone.

**The data:** Thread engagement data consistently shows steep drop-off. Tweet 1 gets ~100% of impressions. Tweet 5 gets ~30-40%. Tweet 10 gets ~10-15%. Your best content must live in tweets 1-4.

**How to avoid:** Front-load your best insight. Tweet 1 = hook. Tweet 2 = most surprising/valuable point. Tweets 3-4 = supporting evidence. Later tweets = additional detail for the dedicated readers.

## 3. Filler Tweets Destroy Thread Quality

**What happens:** Agent writes thread tweets like "Let me explain..." or "Here's the thing:" or "This is important:" — tweets that contain no information and exist only to set up the next tweet.

**Why it matters:** Each tweet in a thread can be quote-tweeted independently. If someone quote-tweets your filler tweet, it makes both of you look bad. Every tweet must deliver standalone value.

**How to avoid:** After writing the thread, delete any tweet where the content could be merged into the previous or next tweet. If a tweet is just a transition, cut it.

## 4. Thread Format on LinkedIn ≠ Thread Format on X

**What happens:** Agent writes an X-style numbered thread (1/, 2/, 3/) and posts it to LinkedIn. Or writes a LinkedIn-style long-form post and splits it into X tweets.

**Why it matters:** LinkedIn threads are single posts with numbered sections and line breaks — not multi-post threads. Posting multiple LinkedIn posts rapidly as a "thread" means each post competes against the others (360Brew penalizes this). X threads are genuinely multi-post.

**How to avoid:** On X, use actual multi-tweet threads. On LinkedIn, write a single post with numbered sections and line breaks to create thread-like flow. Never post multiple LinkedIn posts as a "thread."

## 5. The "🧵 Emoji Crutch"

**What happens:** Agent starts every thread with "🧵" or "Thread 🧵" as if the audience needs warning that it's a thread.

**Why it matters:** If the content is worth reading, people will read it. The 🧵 emoji alone is fine as a visual indicator, but "Thread 🧵" is filler. And on X, overuse of the pattern has been flagged as an AI-tell — it's one of the most common AI-generated thread openings.

**How to avoid:** Start with the hook, not a label. If the hook is strong enough, people will read the thread without being told it's a thread.
