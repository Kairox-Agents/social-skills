# Cross-Posting Rules

> What changes between each platform pair. Never copy-paste across platforms — adapt every time.

---

## Platform Priority Sequencing

Post to your **primary platform first**, then adapt outward. Timing matters.

### Recommended Sequence

```
Primary platform (where your biggest audience lives)
    ↓ same day
Secondary platform (adapted version)
    ↓ 1-2 days later
Tertiary platforms (further adapted)
    ↓ 1 week later
Evergreen reposts (Reddit, community posts, newsletters)
```

**Why stagger:** Each platform's algorithm favors fresh content. Posting the same idea everywhere simultaneously means you compete with yourself for your own audience's attention (many people follow you on multiple platforms).

## Platform Pair Adaptation Checklists

### X → LinkedIn

| Element | X Version | LinkedIn Adaptation |
|---------|-----------|-------------------|
| Length | 280 chars / thread | 800–2,200 chars single post |
| Hook | Punchy, 1 line | Expanded, 2–3 lines (must land before 210-char "see more" fold) |
| Tone | Sharp, informal | Professional-casual, slightly warmer |
| Structure | Short sentences, line breaks | Same line breaks but add context/story |
| Hashtags | 0–1 | 2–3 at the end |
| Emoji | 0–2 max | 0–3, conservative use |
| CTA | "What would you add?" | "What's your experience with this?" |
| Links | In reply (not main tweet) | In first comment (not main post) |
| Thread → single post | Merge into flowing paragraphs | Keep numbered points but add transitions |

### X → Reddit

| Element | X Version | Reddit Adaptation |
|---------|-----------|-----------------|
| Length | 280 chars / thread | 300–800 words |
| Hook | Bold claim | Question or "here's what I found" |
| Tone | Personal brand | Anonymous/humble, value-first |
| Self-reference | "I did X" fine | Minimize — focus on the insight, not you |
| Hashtags | 0–1 | **Remove ALL** |
| Emoji | 0–2 | **Remove ALL** |
| Links | In reply | Only if directly relevant; self-links = ban risk |
| Subreddit | N/A | Choose carefully — each sub has different rules |

### X → Instagram

| Element | X Version | Instagram Adaptation |
|---------|-----------|---------------------|
| Format | Text tweet | Caption + visual (image/carousel required) |
| Length | 280 chars | 500–1,500 chars |
| Hook | 1 line text | Visual hook (first image) + text hook (first line of caption) |
| Hashtags | 0–1 | 5–15 relevant ones at the end |
| Emoji | 0–2 | 3–5, used as section markers |
| CTA | Reply-focused | Save-focused ("📌 Save this for later") |
| Line breaks | Yes | Yes, even more — Instagram captions need heavy whitespace |

### X → TikTok

| Element | X Version | TikTok Adaptation |
|---------|-----------|------------------|
| Format | Written text | Spoken to camera / voiceover |
| Length | 280 chars | 60–90 seconds spoken (150–250 words) |
| Hook | First sentence | First 3 seconds of video |
| Tone | Written-sharp | Conversational, slightly faster pace |
| CTA | In text | Spoken mid-video (not at the end — viewers drop off) |
| Hashtags | 0–1 | 3–5 in caption (discovery-focused) |
| Rewrite depth | Adapt text | Fully rewrite as a spoken script — don't read tweets on camera |

### X → YouTube (Shorts)

| Element | X Version | YouTube Short Adaptation |
|---------|-----------|------------------------|
| Format | Written text | Visual + spoken (show, don't just tell) |
| Length | 280 chars | 45–60 seconds |
| Hook | First sentence | First 3 seconds visual + spoken hook |
| Structure | Claim → support | Hook → problem → insight → payoff |
| CTA | Reply/bookmark | "Subscribe" at the end (after value delivered) |

### LinkedIn → X

| Element | LinkedIn Version | X Adaptation |
|---------|-----------------|-------------|
| Length | 800–2,200 chars | Compress to 280 or break into thread |
| Story | Full narrative arc | Extract the punchline or core insight only |
| Tone | Professional-warm | Sharper, more direct |
| Hashtags | 2–3 | 0–1 |
| "See more" hook | 210 chars | Entire tweet IS the hook |
| Paragraphs | Flowing text | One thought per tweet in threads |

### LinkedIn → Instagram Carousel

| Element | LinkedIn Version | Carousel Adaptation |
|---------|-----------------|-------------------|
| Format | Text post | Visual slides |
| Each paragraph → | Inline text | One slide per key point |
| Word density | 50–100 words/section | 10–20 words/slide |
| CTA | Comment at end | "Save + share" on final slide |
| Branding | Name/profile visible | Add your handle on every slide |

### Reddit → X

| Element | Reddit Version | X Adaptation |
|---------|---------------|-------------|
| Length | 300–800 words | Pull the single best line |
| Tone | Humble/helpful | Add personality back in |
| Attribution | Anonymous community value | "I saw this insight on Reddit and it changed how I think about [X]" |
| Depth | Full explanation | One sharp claim + "here's why" |

### Reddit → LinkedIn

| Element | Reddit Version | LinkedIn Adaptation |
|---------|---------------|-------------------|
| Tone | Community/anonymous | Personal voice, "I" perspective |
| Structure | Problem → analysis → discussion | Story → insight → question |
| Self-promotion | Zero | Light (link in first comment if relevant) |
| CTA | "What do you think?" | "What's been your experience?" |

## Universal Cross-Posting Rules

### DO
1. **Change the hook** every time — even if the body is similar, the entry point should differ
2. **Respect format constraints** — X is 280 chars, Instagram needs visuals, TikTok needs audio
3. **Adjust formality** — Reddit is the most casual, LinkedIn is the most professional, X is in between
4. **Stagger timing** — don't post everywhere simultaneously
5. **Track what works per platform** — a post that flops on X might crush on LinkedIn and vice versa

### DON'T
1. **Don't screenshot tweets for Instagram** — lazy, doesn't perform, looks phoned in
2. **Don't paste the same text everywhere** — each platform's audience expects native content
3. **Don't use the same hashtag strategy everywhere** — hashtags matter on Instagram, are irrelevant on X, and are banned on Reddit
4. **Don't link to other platforms** — "Check out my TikTok" on LinkedIn = algorithm punishment
5. **Don't post a horizontal video to TikTok** or a vertical video to YouTube (long-form) — format mismatch kills engagement

### The 70% Rule
When cross-posting, at least 70% of the content should feel native to the target platform. If you can tell it was originally written for somewhere else, you haven't adapted enough.

| LinkedIn | Warm, professional, storytelling | Medium | 0–3 | 2–3 end | First comment |
| Reddit | Helpful, humble, specific | Low | 0 | 0 | Only if genuinely useful |
