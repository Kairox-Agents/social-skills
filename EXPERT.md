## Methodology

### How Platform Algorithm Data Was Gathered

This skill pack's platform references are sourced from a combination of:

1. **Official platform sources** (🟢 High confidence)
   - X/Twitter open-source algorithm code (GitHub, April 2023 + January 2026 xAI release)
   - Adam Mosseri (Head of Instagram) public statements and Reels explanations
   - Instagram @creators official account guidance
   - YouTube CEO Neal Mohan annual strategic letters (2025, 2026)
   - YouTube Help Center documentation
   - TikTok Creator Portal
   - LinkedIn VP Gyanda Sachdeva statements on engagement pods (November 2025)
   - Reddit official AutoModerator responses and admin statements

2. **Large-scale third-party analysis** (🟡 Medium confidence)
   - AuthoredUp: 3M+ LinkedIn post analysis (360Brew signals, save vs like weights)
   - Buffer: 4M+ Instagram post analysis (format engagement, DM share signals)
   - TheShieldIndex: LinkedIn impression benchmarks by follower count
   - SocialInsider: Cross-platform format engagement data
   - PostEverywhere: X algorithm analysis from open-source code
   - Typefully: X algorithm practical breakdown
   - OutlierKit: YouTube algorithm timeline and ranking factors
   - vidIQ: YouTube 3 ranking signals and 4-layer testing system
   - Socialync: TikTok algorithm 2026 changes, cross-posting 90-day test
   - Botdog: LinkedIn 360Brew analysis

3. **Community consensus** (🔴 Lower confidence, noted when used)
   - r/TheoryOfReddit analysis of ranking mechanics
   - Creator community reports on platform behavior changes
   - Observed patterns without official confirmation

### Evidence Quality Rating System

Every algorithm claim in the sources.md files is rated:
- 🟢 **High** — Official platform source, open-source code, or executive statement
- 🟡 **Medium** — Credible third-party analysis with large sample size (1000+ posts)
- 🔴 **Low** — Anecdotal, community consensus, or small sample observation

Claims with only 🔴 evidence are noted as such and not presented as facts.

### How This Skill Pack Was Built

1. **Phase 1-2:** Repository structure, Python tooling (stdlib only), validation scripts
2. **Phase 3:** Reference libraries — hook formulas, CTA patterns, anti-patterns, cross-posting rules, ideation frameworks. Written from domain expertise, not generated.
3. **Phase 4:** Omni-viral-humanizer with platform-specific viral examples, AI-tell checklist, sourced algorithm data
4. **Phase 5-6:** Testing examples (real viral post patterns, not synthetic), documentation, distribution manifests
5. **Phase 7:** Integration testing — all scripts verified, all references cross-validated
6. **Platform rewrite (March 2026):** Deep research refresh across all 6 platforms with March 2026 data. Every platform reference and algo deep dive rewritten with dated, sourced claims.

### What's NOT in This Skill Pack

- No proprietary data or insider information
- No claims presented as fact without a source
- No recommendations based solely on "best practices" without specific rationale
- No video production guidance (script outlines only — actual video creation is out of scope)
- No paid advertising strategy (organic content only)

### Maintenance Schedule

Platform algorithms change frequently. This skill pack should be reviewed:
- **Quarterly:** Check all claims against latest platform announcements
- **On major updates:** When a platform announces significant algorithm changes
- **Sources.md dates:** Every claim has a date stamp. Claims older than 6 months should be verified
