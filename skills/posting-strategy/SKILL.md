## Posting Frequency Guidelines

### Recommended Frequency by Platform

| Platform | Minimum | Sweet Spot | Maximum |
|----------|---------|------------|---------|
| X/Twitter | 1/day | 2-5/day | ~10/day |
| LinkedIn | 2-3/week | 1/business day | 1/day |
| Reddit | Comments daily | 1-3 posts/week | Not same sub daily |
| Instagram (feed) | 3/week | 3-5/week | 2/day max |
| Instagram (Reels) | 3/week | 3-5/week | 1/day |
| Instagram (Stories) | 2-7/day | 3-5/day | 10/day |
| TikTok | 1/day | 1-3/day | 3/day |
| YouTube (long-form) | 1/month | 1/week | 2/week |
| YouTube (Shorts) | 3/week | Daily | 2/day |
| YouTube (Community) | 2/week | 3-4/week | Daily |

### Frequency Anti-Patterns

- **Posting 5× then going dark for 2 weeks** → Worse than consistent 2×/week
- **Multiple posts per day on LinkedIn** → Posts compete against each other
- **Same content to 5 subreddits at once** → Spam detection, community backlash
- **10 TikToks per day** → Each video competes with your others for the same audience

## Cross-Posting Rules

### What to Cross-Post (With Adaptation)
- Core message / thesis → adapt voice and format per platform
- Educational content → different depth per platform
- Data/insights → different framing per platform

### What NOT to Cross-Post
- Platform-specific content (Reddit questions → LinkedIn, TikTok trends → LinkedIn)
- Content that requires platform-native context (quote tweets, Duets)
- Time-sensitive content after the window has passed on the first platform

### Cross-Post Sequence

When posting the same idea across platforms:

1. **Discovery platforms first** (TikTok, Instagram Reels) — widest reach, shortest content
2. **Conversation platforms second** (X, Reddit) — drive discussion
3. **Authority platforms third** (LinkedIn, YouTube) — deeper treatment
4. **Space out by 2-4 hours minimum** — don't flood everything at once

### Adaptation Checklist per Cross-Post

- [ ] Hook rewritten for platform
- [ ] Tone adjusted (X: punchy, LinkedIn: substantive, Reddit: casual, TikTok: fast)
- [ ] CTA changed to platform-appropriate action
- [ ] Character count within limits
- [ ] Hashtag strategy matches platform rules
- [ ] Format matches platform norms (no Reddit hashtags, no LinkedIn emoji spam)
- [ ] No watermarks from other platforms (TikTok → Instagram)

## Batching Strategy

For efficiency, batch content creation:

**Weekly batch session (~2-3 hours):**
1. Review content calendar for the week (15 min)
2. Write all X posts for the week (30 min)
3. Write LinkedIn posts (30 min)
4. Create Instagram carousel outlines (30 min)
5. Draft TikTok scripts (20 min)
6. Review all content through content-review (20 min)
7. Schedule or save as drafts (15 min)
